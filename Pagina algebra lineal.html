Memorizar los siguientes axiomas:

  • Si (x ∈ V) & (y ∈ V) => x+y ∈ V

  •  x+0 = 0+x = x

  • x+(-x)=0;

  • x+y = y+x

  • Sea α es un escalar. => Si (x ∈ V) => (αx ∈ V)
  • Sean α un escalares. => α (x+ y)=  αx + α y
  • Sean α y β escalares. => (α + β)x=  αx + βx
  • Sean α y β escalares. => α( βx) = (α β)x
  • 1*x=x

8. Transformaciones lineales

Ejemplo de una transformación lineal de rotación de un punto

close all;

clear all

V=[1 0]'

Tetha=0:0.1:4*pi;  % Dos vueltas


for i=1:length(Tetha)

A=[cos(Tetha(i)) -sin(Tetha(i));sin(Tetha(i)) cos(Tetha(i))];% Esta es simplemente una matriz


U=A*V; %%Transformación lineal del vector V en U es una multiplicación de una matriz por un vector


plot([0 U(1)],[0 U(2)],'*')

axis ([-2,2,-2,2]);

pause(0.000001);

endfor

Ejemplo de transformación de rotación de un triángulo

close all;

clear all


x=[0 1 3 0]

y=[0 2 1 0]


M=[x;y]

Tetha=0:0.1:4*pi;  % Dos vueltas


for i=1:length(Tetha)

A=[cos(Tetha(i)) -sin(Tetha(i));sin(Tetha(i)) cos(Tetha(i))];% Esta es simplemente una matriz


U=A*M; %%Transformación lineal del vector V en U es una multiplicación de una matriz por un vector

xrot=U(1,:);

yrot=U(2,:);

plot(xrot,yrot)

axis ([-4,4,-4,4]);

pause(0.000001);

endfor

% Este programa aplica una trnasformación lineal sobre una imagen que como sabemos es una matriz

 %Rotación


theta=0:0.1:4*pi;  % Dos vueltas


for i=1:length(theta)


T = maketform ('affine', [cos(theta(i)) -sin(theta(i)); ...

                           sin(theta(i)) cos(theta(i)); 0 0]); %%Definición de la matriz de transformación


%Transformación lineal de la matriz

[im2 xdata ydata] = imtransform (im, T); %%Aquí se realiza la multiplicación de A*Imagen sólo que debe hacerse de una manera queno explicaremos

pause(0.000001);

endfor
